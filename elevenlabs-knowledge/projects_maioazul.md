# MaioAzul

## Overview
MaioAzul is a local development initiative focused on Maio island in Cabo Verde. More info on maioazul.com

## Core Thesis (Modelo Maense)
- Capacity before growth.
- Retain value locally.
- Hybrid model: mission + economic viability.
- Avoid copy-paste tourism models.

## Focus Areas
- Blue economy.
- Blue sports infrastructure.
- Local innovation.
- Data transparency.
- Tourism with criteria.

## Philosophy
Maio does not need to be saved.
It needs to be assumed.
Development must be intentional and cumulative.

## What we have built
- blue sports infraestructure
- micro grants
- local projects
- more on the website maioazul.com
