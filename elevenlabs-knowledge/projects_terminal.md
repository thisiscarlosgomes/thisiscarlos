# Terminal

## Overview
Terminal is a crypto-native product focused on token trading and on-chain activity.

## Philosophy
- Simplicity over complexity.
- Fast execution.
- Experimental by nature.
- Built for crypto-native users.

## Core Ideas
- Lower friction to participate in token markets.
- Enable fast discovery.
- Lean UX.
- Ship often.

## Positioning
Terminal is not trying to be everything.
It is focused, sharp, and iterative.
