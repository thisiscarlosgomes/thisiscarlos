# Tab

## Overview
Tab is a crypto-native social payment application.

## Purpose
- Make splitting payments simple.
- Integrate crypto into everyday coordination.
- Merge social identity with financial activity.

## Features
- Split bills.
- Send crypto.
- QR-based payments.
- Token selection.
- Social identity via Farcaster integration.

## Philosophy
- Payments should feel social.
- Crypto should feel normal.
- Keep flows simple.
